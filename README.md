# Misinformation Risk Intelligence System

## Overview
This project builds a probabilistic misinformation risk scoring system with explainability and cross-domain robustness evaluation.

## Key Features
- TF-IDF + XGBoost classification
- Risk probability scoring
- SHAP explainability
- Cross-dataset evaluation (general vs COVID domain)
- Streamlit deployment

## Dataset
- Fake & Real News Dataset (Kaggle)
- COVID-19 Fake News Dataset (Kaggle)
